| 术语 | 含义 |
| --- | --- |
| Boot memory | 从全局和渠道专属路径加载的固定启动上下文 |
| Recall | 针对当前任务检索候选记忆 |
| Disclosure | 记忆应该在未来什么场景出现 |
| Domain | 顶级命名空间，例如 `core`、`preferences` 或 `project` |
| Memory path | 记忆的稳定 URI |
| Glossary | 帮助召回找到正确记忆的别名和关键词 |
| Priority | 与召回评分一起使用的重要性信号 |
| Dream | 审阅并改善记忆质量的维护工作流 |
| View | 从记忆内容生成的可搜索表示，例如 gist 或 question |
| Client type | 运行时标识，例如 `codex`、`claudecode`、`openclaw`、`pi` 或 `hermes` |
