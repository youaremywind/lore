第一次安装 Lore 时，从这里开始。

Lore 是一个开源记忆服务器，并提供可选的 Agent 运行时集成。默认安装器会用 Docker Compose 启动服务器，创建托管 PostgreSQL 数据库，并配置支持的 Agent 渠道。如果你已经维护 PostgreSQL，或想把 Lore 跑在另一台主机上，请改用外部服务器和手动配置路径。

按这个顺序进行：

1. 运行快速安装器，或准备自己的 PostgreSQL 实例和服务器 `.env`。
2. 打开 `/setup`，配置 embedding、View LLM 和启动记忆。
3. 用正确的 `client_type` 和 token 连接 Agent 运行时。
4. 在把持久指令移出本地 Agent 文件之前，先验证召回。

安装器默认把本地状态写入 `~/.lore`。如果想换位置，在运行前设置 `LORE_HOME`。
