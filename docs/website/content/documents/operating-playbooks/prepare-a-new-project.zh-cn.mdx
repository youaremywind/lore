## 起始集合

推荐起始集合：

1. `project://project_name/overview` — 项目是什么，放在哪里。
2. `project://project_name/commands` — 安装、开发、测试、构建和部署命令。
3. `project://project_name/constraints` — 架构规则、禁止变更、产品约束。
4. `project://project_name/deployment` — 托管方式、环境变量、发布流程、回滚说明。
5. `project://project_name/glossary` 或每条记忆上的术语 — 产品名、别名、缩写。

写完第一批记忆后，用真实任务 prompt 跑一次召回。如果项目概览没有出现，改善它的 disclosure 和 glossary。
