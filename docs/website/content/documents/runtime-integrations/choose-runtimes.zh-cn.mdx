选择与你日常工具匹配的运行时集成。

| 运行时 | 什么时候使用 | 集成行为 |
| --- | --- | --- |
| Claude Code | 你用 Claude Code 做编码会话 | MCP 工具、`SessionStart` 启动注入、每次 prompt 召回、`CLAUDE.md` guidance 导入 |
| Codex | 你用 Codex CLI/Desktop | MCP 工具、Lore skill、插件注册、hooks feature 和兼容的用户级 hooks |
| Pi | 你用 Pi agent | 扩展 hooks 注入启动 guidance 和每次 prompt 召回 |
| OpenClaw | 你用 OpenClaw | 插件工具、启动健康检查、启动 guidance 和召回注入 |
| Hermes | 你用 Hermes Agent | Python skill/provider，包含 boot、recall 和 memory tools |

只安装你使用的集成：

```bash
curl -fsSL https://raw.githubusercontent.com/FFatTiger/lore/main/scripts/install.sh | bash -s -- --channels codex,claudecode
```

每个 channel 都应该使用同一个服务器 URL 和 token。当客户端通过 MCP 连接时，端点包含 `client_type` 查询参数，Lore 会据此加载客户端专属启动规则，并按渠道跟踪召回使用。
