常见设置问题通常落在一个边界上。

| 现象 | 检查 |
| --- | --- |
| 服务器无法启动 | Docker logs、`DATABASE_URL`、PostgreSQL 健康状态、Redis URL |
| 迁移失败 | 数据库用户权限、`vector` 扩展可用性、使用时的 `pg_jieba` preload |
| `/api/health` 失败 | Web 容器日志和网络端口映射 |
| 浏览器/API 认证失败 | `API_TOKEN`、bearer header、浏览器 session、反向代理 headers |
| setup 无法保存模型配置 | provider base URL、API key、model name、timeout |

Docker 安装时，在包含 `docker-compose.yml` 的目录检查日志：

```bash
docker compose ps
docker compose logs web
docker compose logs postgres
```
