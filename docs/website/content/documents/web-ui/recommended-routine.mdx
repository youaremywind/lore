A practical weekly routine keeps Lore useful.

## Weekly routine

1. Write or update memories immediately after durable decisions.
2. Run a few realistic recall queries for active projects.
3. Fix missing recall by improving disclosure and glossary terms.
4. Back up memory before cleanup or Dream review.
5. Run Dream manually and review suggestions.
6. Remove or split stale broad memories.

Do not wait for the memory graph to become noisy before maintaining it. Small, regular cleanup is safer than rare large rewrites.
