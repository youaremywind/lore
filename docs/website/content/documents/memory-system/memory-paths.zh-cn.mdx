记忆路径应该稳定、短，并直接描述主题。

常用域：

| 域 | 用途 |
| --- | --- |
| `core://` | Agent 通用规则、系统级行为、渠道专属规则 |
| `preferences://` | 用户偏好、协作风格、长期个人设置 |
| `project://` | 项目知识、命令、部署、架构约束 |

好路径示例：

```text
core://agent
core://agent/codex
preferences://user
project://lorehub/commands
project://lorehub/deployment
```

不要把多个无关主题塞进一个宽泛路径。路径越稳定，链接、移动和审阅就越简单。
