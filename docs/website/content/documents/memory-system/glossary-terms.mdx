Glossary terms help recall find a memory when the prompt uses aliases or shorthand.

## What to add

Good glossary terms include:

```text
LoreHub, Cloudflare Pages, deploy hook, docs website, MDX docs
```

Use glossary for names people actually type:

| Add | Avoid |
| --- | --- |
| project names | generic filler words |
| feature names | every word from the content |
| acronyms | obvious stop words |
| user nicknames for tools | unrelated synonyms |
| error names or symptoms | huge pasted logs |

Glossary is especially useful for projects with multiple names, Chinese/English aliases, or tool-specific terms such as `CLAUDE.md`, `AGENTS.md`, `Codex`, and `OpenClaw`.
